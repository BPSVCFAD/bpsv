1 · Project Setup
STEP 1
Create Project
Sets project name, company, department, start & end dates, team members, and holiday list.
Only PM
STEP 2
Define Activity Types (KRA)
Creates work categories like "Community Survey", "Field Visit", "Training Session".
Only PM
STEP 3
Create & Assign Tasks
Creates tasks under the project, sets priority, due dates, KRA type, and assigns to a Project User.
Only PM
WAITING
Awaiting Task Assignment
Project User waits to be assigned tasks by the Project Manager.
Project User
AUTO
Project & Tasks Saved
Project, Activity Types, and Tasks stored in the system. Progress tracking activated.
Auto
🔔 Task Assigned — Notification sent to Project User
2 · Task Execution
MONITOR
Monitor Task Progress
Views overall project % completion, overdue tasks, and comments on the dashboard.
PM · View
ALSO PM
Senior Users Can Also Create Tasks
District & Block Coordinators can create and assign sub-tasks within their scope.
PM + Senior PU
STEP 4
Update Task Status
Changes status: Open → Working → Pending Review → Completed. Adds progress %.
Project User
STEP 5
Add Comments / Notes
Adds field notes or remarks on the task for the manager to review.
Project User
AUTO
Recalculate % Complete
System auto-updates overall project completion % based on all task progress.
Auto · On update
AUTO
Overdue Alert
If a task passes its due date, system notifies the assigned user the next day.
Auto · +1 day
3 · Monthly Timesheet
STEP 6
Create Timesheet Template
Creates a timesheet for each Project User 1–3 days before the reporting month starts.
Only PM
STEP 7
Fill & Submit Timesheet
Records engagement days per task. Tasks auto-fetched. Must submit by the 29th of each month.
Project User · by 29th
AUTO
Monthly Reminder
Reminds users to fill and submit timesheet before the 29th of each month.
Auto · Monthly
📨 Timesheet Submitted — PM notified for review
4 · Approval
STEP 8
Review Timesheet
PM reviews all engagement days, tasks, and activity types logged by the user.
Only PM
STEP 9
Approve or Reject
Approves to finalise, or rejects with comments asking the user to correct and resubmit.
Only PM
STEP 10
Receive Approval Result
Notified: Approved ✅ (finalised) or Rejected ❌ — correct and resubmit with manager comments.
Project User
AUTO
Pending Approval Reminder
Reminds PM to review any pending timesheets 1 day before end of month.
Auto · End of month
